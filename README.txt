OPACU Badminton Club Website

📌 Project Overview
The OPACU Badminton Club website is a responsive web application designed to showcase badminton rankings and provide useful guides for players. The platform aims to promote badminton excellence by offering structured rankings and educational resources in an engaging interface.

---

🎯 Project Goals
- Display badminton player rankings (e.g., Singles, Doubles, Mixed Doubles)
- Provide helpful guides and techniques for improving gameplay
- Deliver a visually appealing and user-friendly experience

---

🛠️ Technologies Used
- HTML5 – Structure of the web pages  
- CSS – Styling, layout, and responsive design 
- Local development server (GitHub)
